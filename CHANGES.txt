Version 2

- add RTL text rendering API: `drawStringRTL(...)`

- add example: `examples/arabic_text/arabic_text.ino`
- add contextual Arabic shaping (isolated/initial/medial/final) for linked letters
- switch numeric rendering to Western `0-9` digits (maps Arabic-Indic input too)
- remove extra inter-character gap for Arabic draw/marquee rendering
- add dedicated lam-alef ligature glyphs (isolated/final) and shaping rule
